
This repository contains exercises and examples helpful when learning Go. Some
of the applications are chats, file parsers, http servers and clients, session
handling, html template rendering, json parsers, hashers, tcp servers and
clients, redis like database server, simple unix utilities, file analyzers, etc.

Feel free to explore and take what you need.

If you want to build all of that from scratch follow the book: [An introduction
to programming in Go](https://www.golang-book.com/books/intro). You can follow
the book with a great [Golang training course in YouTube](https://www.youtube.com/playlist?list=PLSak_q1UXfPrI6D67NF8ajfeJ6f7MH83S).
